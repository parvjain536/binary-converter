# 🔢 Number System Converter (Flask Web App)

A web-based application built using Flask that converts between:

- Decimal ↔ Binary
- Decimal ↔ Hexadecimal
- Binary ↔ Hexadecimal

## 🚀 Features
- Clean UI with CSS
- Input validation
- Error handling
- Multiple number system conversions

## 🛠️ Tech Stack
- Python
- Flask
- HTML/CSS

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
